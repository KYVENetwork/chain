# Parameters

The bundles module contains the following parameters:

| Key           | Type                    | Example |
|---------------|-------------------------|---------|
| UploadTimeout | uint64 (time s)         | 600     |
| StorageCost   | uint64 (tkyve per byte) | 25      |
| NetworkFee    | string (%)              | "0.01"  |
| MaxPoints     | uint64                  | 5       |
